# Claude Code Configuration

## Project Overview
DS_Chapter: Statistical climate downscaling analysis using Jupyter notebooks

## Key Files
- `DS_notebook.ipynb` - Main analysis notebook
- `requirements.txt` - Python dependencies (ipykernel, xarray, dask, etc.)
- `.venv/` - Local Python virtual environment
- `src/` - Source code modules
- `scripts/` - Analysis scripts
- `data/` - Data directory

## Python Environment
- Interpreter: `.venv/Scripts/python.exe`
- Jupyter kernel configured in `.venv`
- Dependencies installed from requirements.txt

## Notebook Kernel
The project uses ipykernel for Jupyter notebook support. Kernel should be auto-detected from the `.venv` environment.

## IDE Settings
- VS Code: Python interpreter pointing to `.venv/Scripts/python.exe`
- Jupyter formatting enabled via ms-python.python
